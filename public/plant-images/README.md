# 植物照片文件夹

## 命名规则

将植物照片放在此文件夹中，按照以下命名规则：

```
{植物ID}-{位置索引}.{扩展名}
```

### 示例

- `p1-0.jpg` - 植物 p1（银杏）的第 1 个位置的照片
- `p1-1.jpg` - 植物 p1（银杏）的第 2 个位置的照片
- `p4-0.jpg` - 植物 p4（大红袍）的第 1 个位置的照片
- `p4-1.jpg` - 植物 p4（大红袍）的第 2 个位置的照片

### 支持的图片格式

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

### 植物ID对照表

| ID | 植物名称 |
|----|---------|
| p1 | 银杏 |
| p2 | 鸡爪槭 |
| p3 | 桂花 |
| p4 | 大红袍 |

### 位置索引

位置索引从 `0` 开始：
- `0` = 第 1 个位置
- `1` = 第 2 个位置
- `2` = 第 3 个位置
- 以此类推...

### 使用说明

1. 将照片按照命名规则放入此文件夹
2. 如果本地存在对应照片，系统会自动使用本地照片
3. 如果本地不存在，系统会使用 `plantsData.ts` 中配置的在线图片URL
4. 照片会在构建时自动包含到项目中

### 注意事项

- 建议图片尺寸：宽度 800-1200px，高度自适应
- 建议图片大小：每张图片不超过 500KB
- 文件名必须严格按照格式，区分大小写
- 如果某个位置的照片不存在，系统会自动使用在线图片作为备用
